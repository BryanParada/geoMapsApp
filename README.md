## Features

* Auto find your location
* Find new places
* Make directions between your location and desired place

## What did i use in this app?

* Connecting an Angular project with a external library (MapBox)
* Use of markers
* Polylines
* Routes
* Directions 
* Distances
* Custom HTTP Client
